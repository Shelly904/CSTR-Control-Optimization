# CSTR Nonlinear Control with AI-Based Optimization

## Overview
Developed a nonlinear control framework for a Continuous Stirred Tank Reactor (CSTR) using backstepping control and AI-based optimization.

## Key Features
- Nonlinear dynamic modeling of CSTR (mass & energy balance)
- Backstepping controller design for setpoint tracking
- Parameter tuning using Randomized Smoothing Finite Gradient (RSFG)
- Simulation of controller performance and stability

## Results
- Achieved stable tracking of concentration and temperature
- Reduced steady-state error and overshoot
- Demonstrated improved controller performance via optimization

## Tech Stack
Python, NumPy, SciPy (ODE solver), Matplotlib

## Highlights
- Bridged process control with AI optimization techniques
- Applied data-driven tuning for nonlinear systems
